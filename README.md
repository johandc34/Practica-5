# MediTime - Práctica 5

Aplicación móvil Android para la gestión de recordatorios de medicamentos en adultos mayores, con seguimiento de tomas, alertas a familiares y apoyo para médicos o cuidadores.
## Estado del proyecto

El proyecto se encuentra en etapa funcional académica. Actualmente cuenta con módulos para autenticación, registro de medicamentos, programación de recordatorios, confirmación de tomas e historial básico.

### Funcionalidades implementadas

- Registro e inicio de sesión de usuarios.
- Registro de medicamentos.
- Programación de recordatorios.
- Confirmación de tomas.
- Historial de cumplimiento.
- Conexión con Firebase.

### Funcionalidades en mejora

- Reportes gráficos de adherencia.
- Mejoras de accesibilidad para adultos mayores.
- Panel más completo para médicos o cuidadores.
- Exportación de reportes.
- Pruebas de usabilidad.
